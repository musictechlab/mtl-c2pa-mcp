### Added
- Initial release of `mtl-c2pa-mcp`
- `c2pa_summary` — human-friendly manifest overview (generator, AI flag, actions, watermarks, signature)
- `c2pa_read` — raw manifest store as JSON
- `c2pa_assertions` — list assertions from the active manifest
- `c2pa_ingredients` — list ingredients used to create the asset
- `c2pa_verify` — signature issuer and validation state
- `c2pa_scan` — directory audit for C2PA coverage and AI provenance
- `c2pa_info` — library version and supported MIME types
